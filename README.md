# Tweet Explorer
(Access at: https://still-stream-52421.herokuapp.com)

## Setup

Install the project dependencies:

`npm install`

Install the fronend dependecies using bower:

`bower install`

## Running

Starts the static and proxy servers:

`npm start`

## Libraries and framework used:
* angularjs
* angular-route
* angular-sanitize
* jquery
* Twitter Bootstrap
* moment
* moment-twitter
